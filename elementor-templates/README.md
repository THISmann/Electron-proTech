# Templates Elementor – proTech Solaire

Import dans WordPress : **Elementor → Templates → Importer** (ou **Bibliothèque → Importer**).

1. Choisir le fichier `.json` concerné.
2. Après import, le template apparaît dans **Mes modèles** ; l’assigner à une section de page (glisser-déposer dans l’éditeur).
3. **Hero** : remplacer l’image placeholder par votre visuel drone/chantier.
4. **Réalisations** : le widget « Posts » doit avoir le type `realisation` (CPT). Ajuster en cas de slug différent.
5. **Formulaire lead** : remplacer `[contact-form-7 id="1" ...]` par l’ID réel de votre formulaire CF7 ou WPForms.

Variantes de wording CTA (à tester en A/B) :
- Audit : « Demander un audit gratuit » / « Obtenir mon audit énergétique » / « Audit solaire gratuit »
- Devis : « Demander un devis » / « Devis sur mesure » / « Simulation gratuite »
