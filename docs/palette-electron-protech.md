# Palette Electron ProTech (logo)

À utiliser dans Elementor, Astra Customizer et tout nouveau bloc pour garder une UI/UX cohérente.

## Couleurs du logo (référence)

Le logo associe feuille verte, éclair jaune, orbes bleues, texte « Electron » (vert) et « ProTech » (bleu). La charte ci-dessous décline ces tons sur le site.

## Couleurs principales (charte site)

| Usage | Nom | Hex | Exemple |
|-------|-----|-----|--------|
| Vert clair (feuille) | `--ept-green-bright` | `#8BC34A` | Accents, highlights |
| Vert moyen | `--ept-green-medium` | `#4CAF50` | Boutons secondaires, bordures |
| Vert foncé | `--ept-green-dark` | `#2E7D32` | Titres, footer, sections nature |
| Bleu clair (électrons) | `--ept-blue-light` | `#03A9F4` | Sous-titres, liens |
| Bleu moyen | `--ept-blue-medium` | `#2196F3` | Liens, CTA secondaires, focus |
| Bleu foncé | `--ept-blue-dark` | `#1565C0` | Texte sur fond clair pour contraste |
| Jaune (éclair) | `--ept-yellow` | `#FFEB3B` | Accents sur fond sombre |
| Or | `--ept-gold` | `#FFC107` | **CTA principal** (audit, devis), éclair du logo |
| Noir | `--ept-black` | `#111111` | Fond hero, barre de navigation, texte sur bouton or |
| Gris foncé | `--ept-gray-dark` | `#37474F` | Texte corps |
| Gris clair | `--ept-gray-light` | `#ECEFF1` | Fonds de section, cartes |
| Blanc | `--ept-white` | `#FFFFFF` | Fonds, texte sur couleur |

## Règles d’usage

- **CTA principal** (audit gratuit, devis) : fond **#FFC107** (or), texte **#111111**.
- **CTA secondaire** (en savoir plus, voir réalisations) : fond **#2196F3**, texte blanc.
- **Sections sombres** (hero) : fond **#111111** ou **#2E7D32**, texte blanc, sous-titres **#03A9F4** ou **#FFEB3B**.
- **Titres H1/H2** : **#2E7D32** (vert foncé).
- **Liens** : **#2196F3** au repos, **#2E7D32** au survol.
- **Focus accessibilité** : outline **#2196F3**.

## Elementor

Dans les widgets (bouton, titre, section), utiliser ces hex directement pour rester aligné avec le thème enfant.
